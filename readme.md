Pinterest Board backup script
====

This script depends on

[casperjs - 1.1.*](https://github.com/n1k0/casperjs/)


Running the script

    $ casperjs pinterest.js --board=http://{url-to-pinterest-board}

All backups are stored in the backup/ directory
